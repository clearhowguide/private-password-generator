# Contributor Code of Conduct

We want Private Password Generator to be a welcoming, respectful, and useful project.

Contributors should:

- use respectful language;
- focus on what is best for users;
- protect user privacy;
- avoid misleading security claims;
- help keep the project clear and safe.

Do not:

- ask users to share real passwords;
- include real passwords in issues or pull requests;
- add features that silently collect sensitive information;
- make claims such as "100% secure" or "unhackable."

Project maintainers may remove comments, issues, pull requests, or contributions that do not follow these standards.
