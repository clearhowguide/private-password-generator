# Contributing to Private Password Generator

Thank you for your interest in contributing.

This project is intentionally small, privacy-first, and educational.

## What we welcome

We welcome improvements to:

- privacy;
- security;
- accessibility;
- password generation quality;
- documentation clarity;
- test coverage;
- maintainability.

## What we do not want

Please avoid changes that:

- send generated passwords to a server;
- store generated passwords in browser storage;
- add analytics around generated passwords;
- log generated passwords;
- use Math.random for password generation;
- add unnecessary dependencies;
- promote misleading security claims.

## Before submitting

Run:

<pre><code>npm run check</code></pre>

Do not include real passwords in issues, screenshots, pull requests, or examples.
